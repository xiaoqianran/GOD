# The Ville Map Package

This is the default GOD pixel-town map package. It follows the v1 map package
layout:

```text
map.yaml
visuals/
  the_ville_jan7.json
  map_assets/**/*.png
characters/
  *.png
```

Use this directory as the reference shape for new map packages. A new map can be
added by copying the template package, replacing `map.yaml`, `visuals/map.json`,
tileset images, and optional character or location assets.
