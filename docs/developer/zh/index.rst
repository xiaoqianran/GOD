GOD 开发者文档
==============

GOD 把本地控制台、FastAPI 后端、Pixel Town 回放环境和 JiuwenClaw Agent runtime 组合成一个可操作的 agent society 产品。这份文档面向想运行、定制或扩展 GOD 仓库的人。

.. note::

   上游 AgentSociety 文档介绍的是 GOD 所依赖的框架。本网站介绍的是 GOD 当前仓库里的产品化集成路径。
