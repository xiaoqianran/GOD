Use cases

What can you do with an agent town?

GOD is built for scenarios people can understand at a glance: ordinary life, public events, targeted questions, and visible interventions.

Observe daily life

Start from a baseline town and watch routines, social ties, errands, and quiet conflicts unfold over time.

Test public events

Inject a campus announcement or local disruption and see who notices, gathers, asks, or ignores it.

Ask anyone

Question one resident, a group, or the whole town during a live run without leaving the map.

Intervene live

Write an instruction for the next step, run it, then compare the social ripple in replay.

Create worlds

Use the browser setup wizard to draft a scenario, generate agents, edit steps, and launch.

Extend the cast and map

Add residents through Agent Studio and publish new local maps through Map Studio.

Current experiments

Start from real scenarios already in the repo.

Each experiment is a reproducible folder with map, cast, setup, and operator notes. New experiments can be added as new cards.

How it feels

Observe first. Ask precisely. Change the next step.

2. Watch the map and resident state update.
3. Ask or intervene from the command rail.
4. Replay the effect step by step.
