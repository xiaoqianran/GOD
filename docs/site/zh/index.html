使用场景

你可以用一座 agent 小镇做什么？

GOD 面向普通人也能理解的场景：日常生活、公共事件、定向提问和可见干预。

观察日常生活

从基线小镇开始，看日程、社交关系、差事和细小冲突如何逐步展开。

测试公共事件

注入校园通知或地方事件，观察谁注意、谁聚集、谁发问、谁忽略。

向任何人提问

在 live run 中向单个居民、群组或全镇提问，不用离开地图。

实时干预

给下一步写入指令，运行后在 replay 中比较社会涟漪。

创建世界

用浏览器向导起草场景、生成 agent、编辑 steps 并启动。

扩展人物和地图

通过 Agent Studio 加居民，通过 Map Studio 发布新的本地地图。

当前实验

从仓库里已经存在的真实场景开始。

每个实验都是可复现目录，包含地图、角色、配置和操作说明。之后新增实验只需要继续补卡片。

体验路径

先观察，精准提问，再改变下一步。

1. 选择或创建一个实验。
2. 观察地图和居民状态更新。
3. 从命令栏 ask 或 intervene。
4. 按 step 回放干预效果。
