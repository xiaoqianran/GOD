校园公共事件实验

PKU Trump Visit

一个风格化校园事件，用于测试通知传播、人群移动、定向提问和操作员干预。

场景

先从普通校园生活开始。随后操作员注入公开访问通知，并把注意力引向百周年纪念讲堂。

地图和角色

实验包含校园 agent 和风格化代表团 persona。这是一个假设性 AI simulation，不是真实行程或真实发言。

推荐玩法

  • 注入访问通知。
  • 询问学生听到了什么、是否会参加。
  • 运行 operator script 并回放人群反应。
